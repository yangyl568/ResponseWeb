###Welcome to use MarkDown
####一个响应式网站

####采用技术：H5 + css + owl.carousel.js插件

	开发流程： 
	1、模仿一个网站布局开始画页面	
	2、使用owl.carousel.js插件 制作轮播图
	3、设置断点(响应式关键点)
	4、根据断点来 调整布局样式(@media)
